3. Como ultimo paso corremos el script de [Insercion - Base de Datos - Petshop.sql](https://github.com/Jemiinat/PETSHOP-MAZA/blob/main/Insercion%20-%20Base$20de$20Datos$20-$20Petshop.sql)
